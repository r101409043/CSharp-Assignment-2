1. What happens when you divide an int variable by 0?
It throws a DivideByZeroException at runtime.
2. What happens when you divide a double variable by 0?
It returns Infinity or -Infinity, depending on the sign.
3. What happens when you overflow an int variable, that is, set it to a value beyond its range?
It wraps around to the other end of the range without an error.
4. What is the difference between x = y++; and x = ++y;?
x = y++: assigns x first, then increases y.
x = ++y: increases y first, then assigns to x.
5. What is the difference between break, continue, and return when used inside a loop statement?
Break: stops the whole loop.
Continue: skips to the next loop iteration.
Eeturn: exits the method completely.
6. What are the three parts of a for statement and which of them are required?
Initialization, condition, and increment.
Only the semicolons are required; the parts can be empty.
7. What is the difference between the = and == operators?
=: is for assignment.
==: is for comparison.
8. Does the following statement compile? for ( ; true; ) ;
Yes, it compiles. It’s an infinite loop with an empty body.
9. What does the underscore _ represent in a switch expression?
It means “default case” or “match anything.”
10. What interface must an object implement to be enumerated over by using the foreach statement?
IEnumerable
